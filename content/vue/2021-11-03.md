개발을 하다보면 동적 데이터 리스트 안에 `Checkbox`가 필요한경우가 종종 있습니다.

아래에 해당 기능과 `Checkbox All`까지 구현한 예제를 간단히 정리해보았습니다.

```html [pages/users.vue]
<template>
  <div class="container">
    <table class="user_table">
      <thead>
        <tr>
          <th><input type="checkbox" :model="selectAll"></th>
          <th>Name</th>
        </tr>
      </thead>
      <tbody>
        <tr v-for="user in users" :key="user.id">
          <td><input type="checkbox" :model="selected" :value="user.id"></td>
          <td>{{ user.name }}</td>
        </tr>
      </tbody>
    </table>
  </div>
</template>

<script>
export default {
  data() {
    return {
      users: [
        {id: 1, name: '철수'},
        {id: 2, name: '영희'},
        {id: 3, name: '길동'}
      ],
      selected: []
    }
  },
  computed: {
    selectAll: {
      get: function() {
        return this.users ? this.selected.length == this.users.length : false;
      },
      set: function(value) {
        this.selected = [];

        if (value) {
          for (user in users) {
            this.selected.push(user.id);
          }
        }
      }
    }
  }
}
</script>
```
