포스팅 내부에 이미지 파일은
~/assets/images/content/ 경로 안에 포스팅과 같은 경로 폴더 생성 후 추가

이미지 태그 example

```
<div><article-image src="/life/place/map.jpg" alt="baobab" /></div>
```

※ article-image 태그만 사용할 시 해당 이미지 아래 모든 블로그 내용이 날아가는 현상 발생, 어쩔 수 없이 div태그를 감싸주어 해결
